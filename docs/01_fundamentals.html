Methods for Data Science

Data Storage

SQL

SELECT * FROM table WHERE condition

Data is stored in a tables; can be difficult to represent complex data

NoSQL

{
  "name": "John McCrae",
  "affiliation": "University of Galway",
  "email": "john@mccr.ae"
}

Flexible data structures; can be difficult to query

Graph Databases

(John McCrae)-[:affiliation]->(University of Galway)
(John McCrae)-[:email]->(john@mccr.ae)

More flexibility but can be very slow to query
